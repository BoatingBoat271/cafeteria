# Café Nómada — Landing Page (Bootstrap)

Landing page **one-page** y **responsive** para una cafetería ficticia llamada **Café Nómada**, desarrollada como proyecto integrador **EF-M2**.  
El sitio utiliza **HTML semántico + Bootstrap 5** para el layout/componentes y **JavaScript (jQuery)** para interactividad: validación de formulario, mensajes controlados por JS, smooth scroll y cierre del menú móvil.

## Enlaces
- Repositorio (GitHub): https://github.com/BoatingBoat271/cafeteria
- Demo (GitHub Pages): https://boatingboat271.github.io/cafeteria/
  
## Tecnologías
- HTML5 (estructura semántica)
- CSS3 (estilos personalizados mínimos)
- Bootstrap 5 (grid, utilidades y componentes)
- JavaScript + jQuery (interactividad)
- Git + GitHub (control de versiones)
- GitHub Pages (publicación)

## Requisitos cumplidos (según pauta)
- Estructura semántica con etiquetas HTML5 (`header`, `nav`, `main`, `section`, `footer`)
- Diseño con **Bootstrap Grid** (`container`, `row`, `col-*`)
- Uso de **componentes Bootstrap** (mínimo 5): Navbar, Botones, Cards, Carrusel, Formularios, etc.
- Interactividad con JS/jQuery:
  - Listener al `submit`
  - `preventDefault()`
  - Validación simple (campos no vacíos)
  - Mensajes de estado controlados por JS
- Responsive: Navbar colapsa en móvil (menú hamburguesa) y layout adaptativo

## Estructura del proyecto
> Puede variar según tu organización, pero la idea base es:

- Raíz del proyecto:
  - `index.html` (archivo principal)
- Carpeta de estilos:
  - `assets/css/style.css` (o `css/style.css`)
- Carpeta de scripts:
  - `assets/js/main.js` (o `js/main.js`)
- Imágenes:
  - `img/` (o `assets/img/`)

## Secciones principales
- Navbar fija (sticky) + colapso en móvil
- Hero con propuesta de valor y CTAs
- Nosotros
- Menú destacado (Cards)
- Galería (Carrusel)
- Para llevar
- Reseñas
- Contacto/Reserva (Formulario)
